Claim a parcel, invite collaborators, and build whatever you can imagine. Save blueprints. Sell them on the marketplace. This is your land.

// RPG Combat

Level up.
Fight everything.

Melee or magic — your choice. Crit chance, mana management, defense scaling. Five enemy types from squishy slimes to dragon-tier drakes with full AI state machines patrolling, chasing, attacking.

Die? Lose 5% gold. Win? Loot drops, XP, and the leaderboard glory. Level cap is 100. Good luck.

// 100% Open Source

Fork it. Host it. Own it.

TypeScript backend. Godot 4.6 client. Every line of code is yours to read, modify, and deploy. Run your own world with your own rules.

vibelife — bash
# Clone and run your own VibeLife server
$ git clone https://github.com/vibelife/vibelife.git
$ cd vibelife && npm install
$ npm run dev

⚡ Server listening on http://localhost:3000
⚡ WebSocket ready at /ws/regions/:id
⚡ 3 regions loaded, 0 players online
⚡ Enemy AI tick started (1s interval)
⚡ Combat regen tick started

# Stack: Fastify 5 + TypeScript + Godot 4.6 + GDScript
# 50+ services • 40+ route files • WebSocket real-time
# Dual persistence: in-memory (dev) / PostgreSQL (prod)
