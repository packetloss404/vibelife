RPG Combat

Level up.
Fight everything.

Melee or magic — your choice. Crit chance, mana management, defense scaling. Five enemy types from squishy slimes to dragon-tier drakes with full AI state machines patrolling, chasing, attacking.

Die? Lose 5% gold. Win? Loot drops, XP, and the leaderboard glory. Level cap is 100. Good luck.

$ cd vibelife && npm install
$ npm run dev
