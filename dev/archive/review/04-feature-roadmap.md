# Feature Roadmap - VibeLife

## Tier 1: Core Experience (Must-Have, Next 3 Months)

### 1. Lo-Fi Radio System
- Ambient music zones tied to parcels or objects
- Owner can set playlist URLs (YouTube/SoundCloud/custom)
- Spatial audio - music fades with distance
- Shared listening with "now playing" display
- Genre channels: lo-fi, jazz, ambient, nature sounds
- **Why:** This IS the vibe. Music is the soul of the lo-fi aesthetic.

### 2. Emote & Expression System
- 20+ emotes: wave, dance, sit, meditate, yoga, stretch, high-five
- Emoji reactions that float above avatar
- Custom emote wheel (radial menu)
- Emote combos (two players high-five when both trigger it nearby)
- Sitting on benches/chairs (contextual interaction)
- **Why:** Social expression is the #1 driver of engagement in virtual worlds.

### 3. Region Biomes & Themes
- Each region has a distinct visual identity:
  - **Aurora Docks:** Harbor sunset, warm oranges, boat sounds
  - **Glass Garden:** Botanical greenhouse, lush greens, water features
  - **Neon District:** Cyberpunk nights, purple/pink neons, rain
  - **Cloud Summit:** Mountain peak, fog, cool blues, wind sounds
  - **Cozy Village:** Autumn town, warm lights, leaf particles
  - **Zen Retreat:** Japanese garden, cherry blossoms, koi pond
- Region-specific ambient sound design
- Region-specific weather patterns
- **Why:** Visual variety drives exploration. Players need reasons to teleport.

### 4. Chat Upgrade
- Chat bubbles above avatars (in-world)
- Whisper (private message to nearby avatar)
- Region-wide announcements
- Chat history persistence
- Emoji support with picker
- Chat channels (general, trade, help, roleplay)
- Typing indicator
- **Why:** Chat is the social backbone. Current implementation is minimal.

### 5. Improved Build System
- Snap-to-grid improvements with visual grid overlay
- Object grouping (select multiple, move as one)
- Copy/paste with Ctrl+C/V
- Undo/redo stack (Ctrl+Z)
- Blueprint system (save/load entire builds)
- Material/color picker for buildable objects
- Object search/filter in build palette
- **Why:** Building IS the game. Make it feel like a creative tool.

## Tier 2: Social & Community (Months 3-6)

### 6. Player Housing System
- Claim a parcel as your "home"
- Home teleport button (always go home)
- Visitors can rate/favorite homes
- Featured homes showcase
- Home privacy settings (public/friends/private)
- Doorbell notification when someone visits
- **Why:** Personal space creates attachment and retention.

### 7. Events System
- Create timed events with descriptions
- Event calendar visible to all players
- RSVP and reminders
- Event-specific builds (temporary stages, decorations)
- Recurring events (weekly dance party, build competition)
- Event leaderboards
- **Why:** Events create appointment viewing and community rituals.

### 8. Marketplace / Economy
- Player-to-player object trading
- Asset marketplace (sell your builds/assets)
- Auction system for rare items
- Price history and trending items
- Creator storefronts (your own shop page)
- Commission system (hire builders)
- **Why:** Economy drives long-term engagement and creator motivation.

### 9. Achievement & Progression
- Explorer badges (visit X regions)
- Builder badges (place X objects, get X visitors)
- Social badges (make X friends, attend X events)
- Collector badges (own rare items)
- Daily/weekly challenges
- XP system with levels (purely cosmetic rewards)
- Title system ("Master Builder", "Social Butterfly", "Explorer")
- **Why:** Progression gives purpose and recognizes investment.

### 10. Guild/Group Enhancements
- Group-owned parcels (collaborative building)
- Group chat channels
- Group events
- Group treasury (shared currency)
- Officer roles with permissions
- Group emblems/banners displayable in-world
- Alliance system (groups can ally)
- **Why:** Groups are the foundation of MMO communities.

## Tier 3: Advanced Features (Months 6-12)

### 11. In-World Scripting (Visual)
- Node-based visual scripting (like Unreal Blueprints lite)
- Trigger zones (enter/exit events)
- Interactive objects (click to activate)
- Timed events (day/night triggers)
- Physics interactions (doors, elevators, platforms)
- Simple state machines
- **Why:** Scripting makes the world feel alive without requiring coding knowledge.

### 12. Voice Chat (Spatial)
- Proximity-based voice (volume decreases with distance)
- Push-to-talk and voice activation modes
- Voice channels per parcel
- Mute/deafen controls
- Voice activity indicators above avatars
- **Why:** Voice makes social interaction 10x more immersive.

### 13. Pets & Companions
- Adoptable pets (cats, dogs, birds, fantastical creatures)
- Pets follow your avatar
- Pet interactions (pet, feed, play)
- Pet customization (colors, accessories)
- Pet tricks and emotes
- Rare pet variants
- **Why:** Pets add emotional attachment and collection mechanics.

### 14. Photography & Media
- In-world camera with filters
- Photo gallery per player
- Screenshot sharing (social feed)
- Photo frames placeable in parcels
- Video screens (play YouTube/streams)
- Projection mapping on surfaces
- **Why:** User-generated content is free marketing.

### 15. Seasonal Content
- Seasonal decorations (holiday themes)
- Limited-time items and rewards
- Seasonal events and competitions
- Region visual changes per season
- Seasonal achievement tracks
- **Why:** Seasonal content creates urgency and return visits.

## Tier 4: Platform Features (Year 2)

### 16. Mobile Companion App
- View friends list and chat
- Manage inventory
- Browse marketplace
- Get notifications
- Simplified 3D world view
- Quick teleport

### 17. Creator Tools Platform
- Blender-to-VibeLife asset pipeline
- Asset review and approval process
- Creator analytics dashboard
- Revenue sharing for marketplace creators
- SDK for custom tools/plugins

### 18. Federation / Multi-Server
- Connect multiple VibeLife instances
- Cross-server teleportation
- Shared identity across servers
- Federated marketplace
- Server directory

### 19. AI NPCs
- Ambient NPCs that populate regions
- Shopkeeper NPCs for marketplace
- Quest-giving NPCs for daily challenges
- Tour guide NPCs for new players
- Customizable NPC behavior via scripting

### 20. VR Support
- Oculus/Meta Quest support
- Hand tracking for building
- Spatial audio in VR
- VR-specific UI
- Comfort options (teleport locomotion, vignette)
