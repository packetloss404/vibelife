# Legacy Review Archive

This directory is kept as a historical review snapshot from an earlier pass over the codebase.

## Archive Status

- These documents are **not** the current source of truth.
- Several claims in this folder are now stale because the codebase evolved after the review was written.
- The authoritative current audit now lives at `docs/audit/index.html`.

## What To Use Instead

- Use `docs/audit/index.html` for the current findings, fixes made, verification status, and open discussion items.
- Use `docs/dev/README.md` for active planning and implementation notes.

## Why This Folder Still Exists

- It preserves earlier architectural thinking and roadmap context.
- It is useful for comparing prior assumptions against the current repository state.
- It should be cited as historical material, not as current operational documentation.
